## 🔧 当前配置

```javascript
const CANVAS_CONFIG = {
  width: 600,                    // 基础尺寸
  height: 600,                   // 基础尺寸
  superResolutionScale: 5,       // 5倍超分辨率渲染
  outputScale: 5,                // 输出也是5倍 = 3000x3000像素
};
```

## 🔄 配置说明

当前逻辑：
- `superResolutionScale: 5` - 5倍分辨率渲染
- `outputScale: 5` - 输出也保持5倍
- 结果：输出3000x3000像素图片

如需切换回600x600：
- 保持`superResolutionScale: 5` - 保持高质量渲染  
- 改为`outputScale: 1` - 缩放回600x600输出
- 结果：600x600抗锯齿图片
